# dati-milano-trasparenza

Questa repository ha come obiettivo quello di condividere dati richiesti al Comune di Milano, o sue partecipate, tramite richieste di accesso generalizzato agli atti ai sensi dell'art. 5 del D.Lgs. 33/2013, come modificato dal D.Lgs. 97/2016, noto come richiesta FOIA.
